# 🌸 Period Tracker Pro

A beautifully designed **Advanced Period Tracker Web App** with animations, dark mode, and cycle history visualization.

---

## ✨ Features

* 📅 Predict next period date
* 🌼 Ovulation tracking
* 💖 Fertile window calculation
* 📊 Cycle history chart (Chart.js)
* 🌙 Dark / Light mode toggle
* 🎨 Smooth animations + modern UI
* 💾 Local storage (saves user data)
